# Webサイトエビデンス取得ツール 📸

Webサイトのテスト作業で取得したスクリーンショット（エビデンス）を、プロジェクト・テストケース単位で管理・整理するためのブラウザベースのツールです。

## 機能

- **クリップボード貼り付け** — `Ctrl+V` / `Cmd+V` でキャプチャ画像を直接貼り付け
- **ファイル選択 / ドラッグ＆ドロップ** — 複数画像ファイルを一括登録
- **タイムスタンプ自動付与** — 貼り付けた画像に取得日時を自動でスタンプ
- **プロジェクト管理** — 複数のプロジェクトをサイドバーで切り替え
- **テストケース管理** — 各プロジェクト内にテストケースを作成し、エビデンスをドラッグ＆ドロップで整理
- **画像編集** — 矩形・線の描画とアンドゥに対応したインラインエディタ
- **ZIP エクスポート** — エビデンスを `Before` / `After` / `Export` フォルダに整理した ZIP ファイルとして一括ダウンロード
- **データ永続化** — IndexedDB にデータを保存し、ブラウザを閉じてもデータを維持

## 動作環境

- Windows（`start_tool.bat` による起動）
- モダンブラウザ（Chrome / Edge など、ES Modules・Clipboard API 対応）

## 起動方法

### Windows（推奨）

`start_tool.bat` をダブルクリックしてください。  
ローカル Web サーバー（ポート 8000）が起動し、ブラウザで自動的にツールが開きます。

```
start_tool.bat
```

### 手動起動

`server.exe` を直接実行することもできます。ポート番号は引数で指定可能です（省略時は 8000）。

```
server.exe 8080
```

その後、ブラウザで `http://localhost:8080/index.html` を開いてください。

### サーバーをソースからビルドする場合

Go がインストール済みの場合は、以下のコマンドでビルドできます。

```
go build -o server.exe main.go
```

## 使い方

1. **プロジェクトを作成する**  
   サイドバーの「プロジェクトの追加」ボタンからプロジェクトを作成します。

2. **テストケースを追加する**  
   「テストケースをまとめて追加」または「＋ 新しいテストケースを追加」ボタンでテストケースを作成します。

3. **エビデンスを貼り付ける**  
   スクリーンショットをコピーし、ページ上の貼り付けエリアで `Ctrl+V` を押すか、画像ファイルをドラッグ＆ドロップします。  
   画像には取得日時が自動的にスタンプされます。

4. **テストケースへ振り分ける**  
   未分類のエビデンスをドラッグ＆ドロップで各テストケースに移動します。

5. **画像を編集する**  
   エビデンス画像の編集ボタンを押すと、矩形や線を描画できる編集モーダルが開きます。

6. **ZIP でエクスポートする**  
   「すべての画像を ZIP 保存」ボタンを押すと、以下の構成で ZIP ファイルがダウンロードされます。

   ```
   プロジェクト名_YYYYMMDD.zip
   ├── Before/
   │   └── テストケース名/
   │       └── 001_コメント.jpg
   ├── After/
   │   └── テストケース名/
   │       └── 001_コメント.jpg   ← 編集済みの場合のみ
   └── Export/
       └── テストケース名/
           └── 001_コメント.jpg
   ```

   「タイムスタンプなしでダウンロード」チェックボックスをオンにすると、スタンプなしの画像でエクスポートされます。

7. **データを保存する**  
   「データを保存」ボタンを押すと IndexedDB にデータが保存されます。  
   ブラウザを閉じても次回起動時にデータが復元されます。

## ファイル構成

| ファイル | 説明 |
|---|---|
| `index.html` | アプリケーションの HTML |
| `main.js` | エントリーポイント・イベントリスナー |
| `state.js` | アプリケーション状態の管理 |
| `actions.js` | プロジェクト・テストケース・エビデンスの操作 |
| `ui.js` | UI の描画ロジック |
| `dom.js` | DOM 要素の参照まとめ |
| `file-handler.js` | ファイル・クリップボードからの画像取り込み |
| `image-editor.js` | 画像編集機能 |
| `zip-export.js` | ZIP エクスポート機能 |
| `indexeddb.js` | IndexedDB への画像保存・読み込み |
| `exif-reader.js` | EXIF データの読み取り |
| `stampWorker.js` | タイムスタンプ付与の Web Worker |
| `utils.js` | 共通ユーティリティ関数 |
| `style.css` | カスタムスタイル |
| `main.go` | Go 製ローカル HTTP サーバー |
| `server.exe` | ビルド済みサーバー実行ファイル（Windows 用） |
| `start_tool.bat` | ツール起動スクリプト（Windows 用） |
